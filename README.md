<h1 style="text-align: center">Lyrics AngularJS</h1>
<h3 style="text-align: center">Simple app with music lyrics</h3>

## Installation
Download or clone repository.

Add .env file with custom PORT field (default 4000)
```
PORT=...
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run watch
```
